## Descripción del Proyecto
El **Encriptador De Texto** es una herramienta básica de encriptación de texto que reemplaza las vocales por los nombres de empresas tecnológicas informáticas. Es una solución simple para proteger documentos de texto de manera básica.

## Estado del Proyecto
El proyecto está **terminado** con posibilidades de futuras actualizaciones.

## Demostración de Funciones y Aplicaciones
El usuario puede escribir un texto y hacer clic en un botón para encriptarlo. También dispone de otro botón que desencripta el texto previamente encriptado. Como aplicación básica, se puede utilizar para resguardar documentos de texto.

## Acceso al Proyecto
El proyecto está disponible para **libre distribución**.
